# Home Security Operations Center (SOC)

## Overview

This project documents a Home Security Operations Center (SOC) built to monitor and investigate real network activity within a home network environment.

The SOC infrastructure runs on a dedicated home server repurposed from an older laptop. The environment integrates multiple security monitoring technologies including Wazuh SIEM, Suricata IDS, pfSense firewall monitoring, and Pi-hole DNS logging.

These tools provide centralized visibility into network activity and enable SOC-style monitoring, alert triage, and incident investigation.

---

## SOC Architecture

The monitoring environment collects telemetry from multiple sources across the network.

Network traffic passes through the pfSense firewall where Suricata IDS analyzes packets for suspicious behavior.

Security events and logs from IDS alerts, firewall activity, DNS queries, and endpoints are forwarded to Wazuh SIEM for centralized monitoring and analysis.

This architecture allows correlation of multiple log sources during alert investigation.

## Security Monitoring Workflow

1. Network traffic is monitored by Suricata IDS.
2. Alerts are generated for suspicious activity.
3. Security events are forwarded to Wazuh SIEM.
4. Additional context is gathered from firewall and DNS logs.
5. Alerts are triaged and investigated using log correlation.

---

## Attack Simulation and Detection

To validate detection capabilities, controlled attack scenarios are performed from a Kali Linux virtual machine running on a Windows laptop within the same network.

These activities simulate real-world attacker behavior and allow investigation of alerts generated by the SOC monitoring stack.

Examples include:

Port scanning reconnaissance  
Brute-force authentication attempts  
Suspicious DNS activity

Each scenario is investigated using SOC-style alert triage and log correlation within Wazuh SIEM.

---

## Project Objective

The objective of this project is to simulate real-world SOC monitoring workflows by detecting, investigating, and documenting security events across multiple telemetry sources.
